# Data Science Portfolio Template

![Portfolio Demo](/assets/img/demo.png)

A modern, responsive portfolio template designed specifically for data scientists and analysts. This template showcases your projects, skills, and experience in a clean, professional layout.

## 🌐 Live Demo
View a live demo of this portfolio at: [https://zanjeel.github.io](https://zanjeel.github.io)

## ✨ Features

- Clean and modern design
- Responsive layout for all devices
- Project showcase with images and descriptions
- Skills and experience sections
- Education timeline
- Contact information
- Easy to customize and deploy

## 🛠️ Built With

- Jekyll - Static site generator
- Markdown - For content management
- HTML/CSS - For styling and layout
- GitHub Pages - For hosting

## 🚀 Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:
- Ruby (version 2.5.0 or higher)
- RubyGems
- Bundler
- Git

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/zanjeel/data-science-portfolio.git
   cd data-science-portfolio
   ```

2. **Install dependencies**
   ```bash
   bundle install
   ```

3. **Run locally**
   ```bash
   bundle exec jekyll serve
   ```

4. **View your site**
   Open your browser and navigate to `http://localhost:4000`

## 📝 Customization

1. **Update Personal Information**
   - Edit `_config.yml` to update your name, title, and social links
   - Modify `index.md` to add your projects, skills, and experience

2. **Add Projects**
   - Place project images in the `assets/img/` directory
   - Update project details in `index.md`
   - Add project links and descriptions

3. **Customize Styling**
   - Modify CSS in the `<style>` section of `index.md`
   - Adjust colors, spacing, and layout as needed

## 🎨 Project Structure

```
data-science-portfolio/
├── _config.yml          # Site configuration
├── _layouts/            # Layout templates
├── assets/             # Static assets
│   └── img/           # Project images
├── index.md           # Main content
└── README.md          # This file
```

## 📦 Required Files

1. **Project Images**
   - Place all project images in `assets/img/`
   - Recommended size: 400x400 pixels
   - Format: JPG or PNG

2. **Configuration**
   - Update `_config.yml` with your information
   - Set your GitHub username and repository name

## 🔧 Troubleshooting

If you encounter any issues:

1. **Ruby Version**
   ```bash
   ruby -v  # Should be 2.5.0 or higher
   ```

2. **Bundle Install Issues**
   ```bash
   gem install bundler
   bundle install
   ```

3. **Jekyll Serve Issues**
   ```bash
   bundle exec jekyll clean
   bundle exec jekyll serve
   ```
